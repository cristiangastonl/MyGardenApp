- Override note color: `textSecondary` should pass WCAG AA on `card` background — if the override note is hard to read on white-ish card background, route to Phase 23 POLISH (which addresses textSecondary contrast). Phase 14 itself does NOT block on this.
- Custom-plant fallback ¿Por qué? hidden: if the section header IS visible (with no body) for a custom plant, FAIL — locked CONTEXT decision is "ENTIRE section hidden including header".
- Voseo / tone: ES strings should feel natural (not academic, not Castilian). If user reports any string sounds wrong, log it and route to Phase 23 POLISH (per-string copy refinement).
- Section content lazy-measure glitch (Pitfall 4): if user reports a 1-frame "0-height flash" before sections settle on first modal open, route to deviation — may need to upgrade EducationalSection to off-screen-measurer pattern (Option B in Pitfall 4).
