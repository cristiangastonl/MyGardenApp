- App boot crash: re-check `app.json newArchEnabled: true` + Metro cache (`npx expo start --clear`) + full clean rebuild (`rm -rf ios/ android/ && npx expo prebuild && npx expo run:ios/android`).
- Sheet gesture failure: re-verify `<GestureHandlerRootView style={{ flex: 1 }}>` shape in App.tsx; check `RNGestureHandlerModule` registration in native logs.

- Skeleton stuttering: verify babel-preset-expo is auto-managing the worklets plugin; check Metro cache.
- PaywallModal Z-order glitch: very unlikely per native Modal behavior; deferred to Phase 14 deviation if it occurs.
